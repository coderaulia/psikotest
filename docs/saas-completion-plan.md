# SaaS Completion Plan

This document tracks the SaaS-first completion order for the platform.

## Current Focus

The product is moving from MVP workspace operations into a production-ready SaaS billing and delivery model.

## Completed Foundations

- customer signup and workspace onboarding
- assessment setup and dummy checkout flow
- participant invitations and reminders
- customer-safe result pages and exports
- teammate access and workspace roles
- upgrade guidance based on usage pressure

## Remaining Major SaaS Tracks

### Billing foundation

- provider-ready subscription schema
- checkout session history
- invoice history
- webhook event storage
- usage events and snapshots

### Commercial flow

- clearer pricing and entitlements
- billing contact management
- self-serve upgrade lifecycle
- later real provider integration

### Participant operations

- bulk participant import/export templates
- invite delivery status
- reminder history

### Customer team operations

- real teammate activation and password setup already exists
- later add invite acceptance analytics and seat lifecycle management

### Reporting and exports

- customer-safe result exports exist
- later add PDFs and release-aware report packaging

## Recommended Order

1. billing foundation
2. customer billing UI enrichment
3. participant import and delivery tracking
4. real provider integration
5. white-label provisioning overlay
